📊 Calculadora de IMC
  Este projeto é uma Calculadora de IMC (Índice de Massa Corporal) desenvolvida com HTML, CSS e JavaScript puro. O objetivo é permitir que o usuário insira sua altura e peso e receba como resultado o seu IMC, além de uma classificação de acordo com a tabela da Organização Mundial da Saúde.

🧮 Funcionalidades
Entrada de altura e peso

  Cálculo automático do IMC com 1 casa decimal

  Exibição do resultado com classificação da condição corporal

  Tabela de referência dinâmica gerada com JavaScript

  Estilo visual moderno e responsivo

  Botão de limpar e de voltar para refazer o cálculo

💻 Tecnologias utilizadas
  HTML5: estrutura da página

  CSS3: estilização e responsividade

  JavaScript (Vanilla): lógica de cálculo, manipulação do DOM e interações

 📌 Observações
O projeto trata entradas com vírgula (ex: 1,75) e converte para ponto.

Classes de cor indicam a situação de saúde: verde (bom), amarelo (alerta), laranja/vermelho (risco).
